Libraries installed -- 
For Linux -
selenium = sudo apt install python3-selenium
pymongo = sudo apt install python3-pymongo
requests = sudo apt install python3-requests
flask = sudo apt install python3-flask

For Windows -
selenium = pip install selenium
pymongo = pip install pymongo
requests = pip install requests
flask = pip install flask

Also ChromeDriver installed
You can Install ChromeDriver for your Linux or Windows From -> 
https://googlechromelabs.github.io/chrome-for-testing/
according to the device specifications.

To run file -
    In Windows - 
        Run python app.py in Command Prompt.
    If Linux - 
        Run python3 app.py in terminal
